## How to use this

1. Open the `notes/` folder in VS Code.
2. Install the **Markdown Preview Enhanced** or use built-in `Ctrl+Shift+V` to preview `.md` files nicely.
3. Work through HTML topics 01→20, then CSS 01→19.
4. Use the `Examples/` files to actually run code in a browser (right-click → "Open with Live Server").
5. Finish with `Exercises/` to apply everything.
